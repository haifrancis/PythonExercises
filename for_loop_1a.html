Python Exercises - Basic For Loops

Group 1:

  • 1️⃣ Print numbers from 1 to 10
  • 👉 Write a Python program that prints numbers from 1 to 10 using a for loop.
  • 2️⃣ Print "Hello, Python!" 5 times
  • 👉 Write a Python program that prints "Hello, Python!" five times using a for loop.
  • 3️⃣ Print even numbers from 2 to 20
  • 👉 Write a Python program that prints all even numbers from 2 to 20 using a for loop.

Group 2:

  • 4️⃣ Print numbers in reverse order
  • 👉 Write a Python program that prints numbers from 10 to 1 in reverse order using a for loop.
  • 5️⃣ Sum of first 10 natural numbers
  • 👉 Write a Python program that calculates the sum of numbers from 1 to 10 using a for loop.
  • 6️⃣ Print the multiplication table of a number
  • 👉 Ask the user for a number and print its multiplication table from 1 to 10 using a for loop.

Group 3:

    • *

*****
  • 8️⃣ Find the factorial of a number
  • 👉 Ask the user for a number and find its factorial using a for loop.
  • 9️⃣ Reverse a word
  • 👉 Ask the user to enter a word, and print the word in reverse order using a for loop.
  • 🔟 Count vowels in a word
