Python Exercises - Basic For Loops

Group 1 :

1️⃣ Print odd numbers from 1 to 19

👉 Write a Python program that prints all odd numbers from 1 to 19 using a for loop.

2️⃣ Print numbers from 1 to N

👉 Ask the user for a number N and print numbers from 1 to N using a for loop.

3️⃣ Print squares of numbers from 1 to 10

👉 Write a Python program that prints the square of numbers from 1 to 10 using a for loop.

Group 2 :

4️⃣ Count from 10 to 100 in steps of 10

👉 Write a Python program that prints numbers from 10 to 100, increasing by 10 each time.

5️⃣ Sum of even numbers from 1 to 20

6️⃣ Reverse a list of numbers

👉 Given a list of numbers [10, 20, 30, 40, 50], write a Python program that prints the numbers in reverse order using a for loop.
